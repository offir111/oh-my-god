התקנה באייפון (Apple) — מקביל להוראות סמסונג / APK
==================================================

חשוב לפני הכל
--------------
בניגוד לאנדרואיד: אין קובץ „APK יחיד” שמתקין מאפס על כל האייפונים בלי מפתח אפל או ללא חנות.
אפל מאפשרים התקנה נוחה בעיקר דרך:
  א׳ הוספת האתר למסך הבית (כמו אפליקציה, ללא Xcode) — הכי דומה ל„קובץ אחד והכל עובד”.
  ב׳ בנייה מלאה ב-Xcode על מק → התקנה למכשיר או IPA לחברים (דורש חשבון מפתח לפי הסוג).


מה כבר מוכן בפרויקט
---------------------
• תיקיית ios/ (Capacitor) — נפתח ב-Xcode על מחשב Mac.
• manifest למובייל ב-client/public/manifest.webmanifest והגדרות ב-client/index.html — מתאימות להוספה למסך הבית.
• client/.env.production — רק כתובת שרת ציבורית (בלי סיסמאות).


דרך מהירה — „לחיצה אחת” מבחינת המשתמש (בלי מק ולא Xcode)
--------------------------------------------------------
זה לא קובץ .ipa אלא התקנה רשמית של האפליקציה כ-Web App מהדפדפן:

1. ודא שהאתר רץ ב־HTTPS (למשל השרת ב-Railway).
2. באייפון: פתח את האתר ב־Safari (לא כרום לצורך ההוספה הזו).
3. הקש על כפתור השיתוף (הריבוע עם החץ למעלה).
4. גלול ובחר „הוסף למסך הבית”.
5. תן שם ואשר — נוצר אייקון על המסך כמו אפליקציה.

עדכון גרסה אחרי שינוי באתר: סגור את האפליקציה מהמסך, פתח שוב את Safari לאותה כתובת ומשוך לרענון או המתן לטעינת הקוד החדש (בדרך כלל מספיק כניסה מחדש).


בניית אפליקציה נייטיבית מלאה (Mac + Xcode)
------------------------------------------
דרוש מחשב Mac עם Xcode מה-App Store.

1. התקן Xcode ואז CocoaPods (למשל: brew install cocoapods).

2. בשורש המאגר:
   npm install
   npm run ios:sync

3. התקן Pods ופתח את הפרויקט:
   cd ios/App
   pod install
   open App.xcworkspace

4. חתימה (Signing): בחר צוות Apple שלך תחת Signing & Capabilities.

5. להתקנה על האייפון שלך בכבל:
   חבר את המכשיר → בחר אותו למעלה ב-Xcode → Run (⌘R).

6. קובץ התקנה להעברה לאחרים (IPA):
   Product → Archive → Distribute App.
   להפצה משפחתית נפוצות: Ad Hoc או Development (דורש רישום מכשירים) או TestFlight דרך App Store Connect (דורש מנוי Apple Developer בתשלום).

עדכון אחרי שינוי קוד בוויב
---------------------------
   npm run ios:sync
   ובנה מחדש ב-Xcode.


סיכום מול סמסונג
-----------------
• סמסונג / אנדרואיד: ראה INSTALL_ANDROID.txt — קובץ APK אחד אפשר לבנות ולהעביר (כולל עם CI ב-GitHub).
• אייפון: ההכי דומה ל„קובץ אחד והתקנה פשוטה” למשתמש קצה הוא „הוסף למסך הבית”; IPA דורש מק וחשבון מפתח.
